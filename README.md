## Key Features

- Make a search with autocomplation of a city.
- Get the current weather.
- Get the forecast weather for 5 days.
- Get an image depending on the city you made the search.

## Built with

- Vite
- React
- Tailwind CSS

## API's

- GeoDB Cities API
- OpenWeatherAPI
- Unsplash API
